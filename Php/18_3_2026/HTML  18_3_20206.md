# Etiquetas de texto

las etiquetas son recursos que sirven para modificar los fragmentos de texto. La mayoria se abren y se cierran con 
Las etiquetas se pueden combinar en un fragmento o linea de texto, siempre que se respeten los ordenes de apertura y cerrado.


Las etiquetas pueden tener atributos que pueden ser obligatorios o no.


b. Es la etiqueta para poner el texto en negrita

br.  es un salto de linea.

I.  Cursiva o Italic.

p.  divide el texto en parrafos

h1-6.  sirven para establecer encabezados, cambiar el tamaño de texto.

center. Centra el texto

img. pone una imagen. pide enlace o ruta y nombre. se puede agregar el parametro "width" para cambiar el tamaño en porcentajes.

ol. se utiliza para definir una lista

li. se utiliza para colocar una entrada en la lista, con numeros que la ordenan

a. Anclar documento, poner enlaces. atributo href: pide un link, pueden ser enlaces o documentos.  Target: se utiliza junto con "_blank" para abrir el enlace en otra ventana.

span. se usa para marcar un fragmento de texto para aplicarle otras etiquetas

div. sirve para dividir por secciones los textos de la pagina,

"table" Se utiliza para definir tablas, con tr define lineas, th pone en negrita el contenido y td permite poner el contenido dentro de tr.
