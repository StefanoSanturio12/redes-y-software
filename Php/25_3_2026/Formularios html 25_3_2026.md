# formularios html


es el input de texto que un usuario puede realizar, con esos datos se pueden enviar al servidor y procesarlo

Todos estos campos se definen con la etiqueta "input type="tipo""
- Text
- number
- password
- date
- mail
- checkbox
- selection
	Este se define con su propia etiqueta;"select" y dentro lleva subetiquetas de "option" que determina las opciones
- radiobutton
- submit
- reset
- button
	""button role=""> boton </button""

Los formularios se crean con la etiqueta form, que tiene 2 atributos

- "Action"
	contien una url que sera donde se enviaran los datos

- "Method"

- "post"
	Envia los datos de forma cifrada



Todos los inputs deben estar en el mismo formulario a la hora de enviarlos y todos deben tener nombre e id.
