# Definiciones

 - *Dato*
	Los datos se refieren a hechos, cifras, simbolos o detalles brutos sin procesar y que representan algo del mundo real. 
		- Puntos claves
			- ==Crudos y sin procesar:== Son la materia prima de cualquier tipo de analisis o interpretaciòn.
			- ==Representacion:== sirve para representar entidades, eventos, caracteristicas o ideas del mundo real
			- ==variedad de formas==: Existen multiples formatos para representarlos, pueden ser digitales o analogicos
			- ==individuales o conjuntos==: Pueden contener un unico valor o una coleccion de valores
			- ==sin contexto inmediato==: por si solos carecen de significado, necesitan ser procesados y contextualizados para convertirse en algo util
		
		 Ejemplos de datos:
		- La temperatura de hoy a las 15PM fue de 30º
		- El nombre de un cliente es "ana perez"
		- El precio de un producto es $10
